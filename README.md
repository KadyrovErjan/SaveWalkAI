# 🦯 SaveWalk AI — Object Distance Voice Assistant

> Система реального времени для обнаружения объектов, определения расстояния до них и голосового оповещения. Создана для помощи слабовидящим людям при ходьбе.

---

## 📸 Как это работает

```
Камера → YOLO (детекция) → Расчёт расстояния → Голосовое оповещение
```

Система захватывает видео с камеры, распознаёт объекты через нейросеть YOLO, рассчитывает расстояние до каждого объекта и **озвучивает предупреждение**, если объект оказывается слишком близко.

## 🗂 Структура проекта

```
SaveWalkAI/
│
├── main.py              # 🚀 Главный файл запуска
├── camera.py            # 📷 Захват видео с камеры
├── detection.py         # 🧠 Детекция объектов через YOLO
├── distance.py          # 📏 Расчёт расстояния
├── voice.py             # 🔊 Голосовое оповещение
│
├── sounds/              # 🎵 Папка со звуковыми файлами
│   ├── person/
│   │   ├── 1.wav        # "1 метр до человека"
│   │   ├── 2.wav        # "2 метра до человека"
│   │   └── 3.wav
│   ├── cell phone/
│   │   ├── 1.wav
│   │   └── 2.wav
│   └── traffic light/
│       ├── 1.wav
│       └── 2.wav
│
├── yolov8n.pt           # Модель YOLO (автозагружается)
└── req.txt              # Зависимости
```

## ⚙️ Установка

### 1. Клонируй репозиторий

```bash
git clone https://github.com/KadyrovErjan/SaveWalkAI.git
cd SaveWalkAI
```

### 2. Создай виртуальное окружение

```bash
python -m venv .venv

# Windows
.venv\Scripts\activate

# macOS / Linux
source .venv/bin/activate
```

### 3. Установи зависимости

```bash
pip install -r req.txt
```

### 4. Добавь звуковые файлы

Помести свои `.wav` файлы в папку `sounds/` согласно структуре выше.  
Каждый файл называется по номеру метра: `1.wav`, `2.wav`, `3.wav` и т.д.

---

## 🚀 Запуск

```bash
python main.py
```

Нажми **`Q`** для выхода.

---

## 🧩 Описание модулей

| Файл | Назначение |
|------|-----------|
| `main.py` | Главный цикл: захват кадра → детекция → отображение → озвучка |
| `camera.py` | Подключение к камере через OpenCV |
| `detection.py` | Запуск YOLO, фильтрация нужных классов объектов |
| `distance.py` | Расчёт расстояния по ширине bounding box |
| `voice.py` | Воспроизведение wav-файла нужного метража в отдельном потоке |

---

## 🎯 Распознаваемые объекты

| Объект | Озвучивается |
|--------|-------------|
| 👤 Человек (`person`) | ✅ Да |
| 📱 Телефон (`cell phone`) | ✅ Да |
| 🚦 Светофор (`traffic light`) | ✅ Да |

> Голосовое оповещение срабатывает если объект **ближе 3 метров**.
