## 📌 О проекте

**SaveWalk AI** — это локальная real-time система, разработанная для помощи людям с ограниченными возможностями зрения при передвижении в городской среде.

Система анализирует видео с камеры, обнаруживает объекты, оценивает их опасность и выдаёт голосовые навигационные команды через заранее записанные `.wav` файлы. Никакого TTS, никакого интернета — всё работает локально.

### Чем это отличается от обычной озвучки

| Обычная система | SaveWalk AI |
|---|---|
| "Человек, 1 метр, справа" | "Идите левее" |
| "Машина, 2 метра, слева" | "Препятствие, остановитесь" |
| "Светофор красный" | "Стойте" |
| Перечисляет всё подряд | Говорит только важное |

## 🧠 Как это работает

```
Камера
  │
  ▼
YOLO + ByteTrack          → обнаружение и трекинг объектов
  │
  ▼
Distance (Pinhole модель) → расстояние до объекта
  │
  ▼
EMA Smoother              → сглаживание скачков
  │
  ▼
Motion Tracker            → approaching / leaving / stable
  │
  ▼
Risk Score                → car > bus > moto > person
  │
  ▼
Top Threat                → выбор одного самого опасного
  │
  ▼
Navigation Logic          → решение: go / stop / left / right
  │
  ▼
Sound Service             → воспроизведение .wav команды
```

## ✨ Возможности

### Обнаружение и трекинг
- Детекция объектов через **YOLOv8n** (person, car, bus, motorcycle, bicycle, traffic light и др.)
- Трекинг через **ByteTrack** с уникальным `track_id` для каждого объекта
- Запуск YOLO не на каждом кадре — экономия CPU/GPU

### Расстояние и движение
- Расчёт расстояния через **pinhole модель** по ширине bounding box
- **EMA-сглаживание** — убирает резкие скачки (0.7→1.0→0.8 → плавная кривая)
- **MotionTracker** — определяет `approaching / leaving / stable` за последние 1.5 сек

### Система риска
- Каждому объекту присваивается `risk_score` на основе класса, расстояния и движения
- Выбирается **один** самый опасный объект (`top_threat`)
- Приоритет: `car` > `bus` > `motorcycle` > `bicycle` > `person`

### Голосовая навигация
- Говорит только **одну команду** за раз
- **Cooldown** 2.5 сек — никакого спама
- Не повторяет одинаковую команду подряд
- Определяет свободную сторону и предлагает обход

### Светофор
- Распознаёт цвет через **HSV-анализ** bounding box
- Озвучивает только при смене цвета или раз в 5 сек

## 🔊 Логика голосовых команд

```
Приоритет   Ситуация                              Команда
──────────────────────────────────────────────────────────────
    1       Объект ближе 1.0м                   🔴 stop.wav
    2       Транспорт приближается < 2.5м        ↙ left.wav / right.wav
    3       Человек по центру < 2.0м            ⚠ person_ahead.wav
    4       Любое препятствие по центру < 3.5м  ↙ left.wav / right.wav
    5       Путь свободен (раз в 5 сек)         🟢 go.wav
```

### Примеры работы

```
Ситуация                          Команда системы
─────────────────────────────────────────────────
Человек идёт навстречу по центру  → "Человек впереди" → "Идите правее"
Машина приближается слева         → "Остановитесь"
Путь свободен 5+ секунд           → "Идите прямо"
Светофор переключился на зелёный  → "Зелёный"
Объект вплотную (< 1м)            → "Стоп"
Все стороны заняты                → "Нет прохода"
```

## 🏗️ Архитектура

```
SaveWalkAI/
│
├── main.py                        # главный цикл
├── camera.py                      # захват кадров
├── sound_manager.py               # низкоуровневое воспроизведение .wav
│
├── core/
│   ├── detector.py                # YOLO + ByteTrack
│   ├── distance.py                # pinhole расчёт расстояния
│   ├── tracker.py                 # EMA, направление, движение
│   └── traffic_light.py          # HSV-анализ цвета светофора
│
├── services/
│   ├── danger_service.py          # risk score + pick_top_threat()
│   ├── navigation_service.py      # навигационные подсказки
│   └── sound_service.py          # логика выбора и воспроизведения звука
│
└── sounds/
    ├── system/                    # go, stop, left, right, no_way
    ├── obstacles/                 # person_ahead, obstacle, very_close
    ├── traffic light/             # red, green, yellow
    └── warning/                   # danger, alert
```

## 🚀 Установка

### 1. Клонировать репозиторий

```bash
git clone https://github.com/yourname/SaveWalkAI.git
cd SaveWalkAI
```

### 2. Создать виртуальное окружение

```bash
python -m venv .venv

# Windows
.venv\Scripts\activate

# Linux / macOS
source .venv/bin/activate
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Скачать модель YOLO

```bash
# Автоматически скачается при первом запуске, или вручную:
python -c "from ultralytics import YOLO; YOLO('yolov8n.pt')"
```

### 5. Запустить

```bash
python main.py
```

Нажми **Q** для выхода.

## ⚙️ Настройка

### Камера

В `camera.py` измени индекс:

```python
cap = cv2.VideoCapture(1, cv2.CAP_DSHOW)
# 0 = встроенная, 1 = USB-камера
```

### Калибровка расстояния

В `core/distance.py`:

```python
FOCAL_LENGTH = 756   # подбери под свою камеру
```

**Как откалибровать точно:**

```
1. Встань ровно на 2.0м от камеры
2. Запусти main.py и посмотри в консоль:
   person #1 | dist=0.8 | ...
3. Посчитай: FOCAL_LENGTH = (pixel_w_bbox * 2.0) / 0.45
4. Вставь результат в distance.py
```

### Пороги навигации

В `services/sound_service.py`:

```python
VERY_CLOSE_M  = 1.0   # ближе → немедленный stop
DANGER_DIST_M = 2.5   # транспорт приближается → предупреждение
PERSON_DIST_M = 2.0   # человек по центру → person_ahead
NAV_DIST_M    = 3.5   # объект по центру → предложить обход
COOLDOWN_SEC  = 2.5   # минимальный интервал между командами
```

## 🖥️ Визуализация

| Элемент | Значение |
|---|---|
| ⬜ Белая рамка | Top Threat — самый опасный объект |
| 🟥 Красная рамка | Объект ближе 3м |
| 🟩 Зелёная рамка | Объект дальше 3м |
| 🟦 Голубой текст | Навигационная подсказка на экране |
| Цветная рамка TL | Цвет сигнала светофора |

Текст на рамке:
```
person #4  1.7m center approaching  r=9
```
`r` — risk score объекта.
